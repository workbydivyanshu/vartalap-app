# Changelog

## v0.1.0 — 2026-03-31

### Features
- Real-time messaging via WebSocket
- Server and channel management
- Direct messages
- Message reactions with 100+ emoji picker
- Reply and thread system
- Message edit and delete
- Message search
- Pin messages
- File upload with progress indicator
- Slash commands (/shrug, /me, /tableflip, /unflip, /lenny)
- Typing indicators
- Desktop notifications
- Quick switcher (Ctrl+K)
- User profile view
- Server invite system
- Settings with persistence
- PWA support (manifest, service worker)
- Mobile responsive design
- Accessibility (ARIA labels, skip link)
- Landing page with modern design
- Help center, download page, terms, privacy pages

### Technical
- React 18 + TypeScript + Vite
- Node.js + Express + Socket.IO
- MongoDB with Mongoose
- JWT authentication
- 46 unit tests (Vitest + React Testing Library)
- Virtual scroll for messages
- Code splitting (vendor chunks)
